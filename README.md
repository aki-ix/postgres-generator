## Назначение

Инструмент позволяет создавать нагрузку на PostgreSQL  
и наблюдать, как база ведёт себя при различных сценариях.

Подходит для:
- тестирования производительности
- анализа роста таблиц
- экспериментов с нагрузкой

---

## Возможности

- Подключение к PostgreSQL (libpq)
- Автоматическое создание тестовой таблицы
- Генерация случайных данных
- Проверка существования таблицы
- Отслеживание размера таблицы в реальном времени

---

## Режимы работы

###  1. Подготовка таблицы (Prepare)

- Заполняет таблицу данными
- Работает до достижения заданного размера

---

###  2. Нагрузочное тестирование (Load test)

Типы нагрузки:
- INSERT
- SELECT
- MIXED

Условия остановки:
- по времени
- по количеству итераций

---

## После выполнения теста выводится отчёт:

- Общее время выполнения
- Количество выполненных запросов
- Производительность (запросов/сек)
- Итоговый размер таблицы
